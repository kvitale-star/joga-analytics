# Frequently Asked Questions

Common questions and answers about using JOGA Analytics.

## General Questions

### What is JOGA Analytics?
JOGA Analytics is a web-based platform for coaches to visualize and analyze match data. It provides interactive charts, detailed statistics, and AI-powered insights.

### Who can use JOGA Analytics?
- **Coaches**: Enter data and analyze performance
- **Administrators**: Manage users, teams, and system settings
- **Viewers**: View data and charts (read-only)

### Do I need special software?
No, JOGA Analytics runs in your web browser. No downloads or installations required.

## Data Entry

### How do I add match data?
Navigate to **Upload Game Data** and fill in the form. See [Upload Game Data](Upload-Game-Data) for detailed instructions.

### Can I upload data from files?
Currently, data is entered manually through the form. Image upload (OCR) is available for some sections to extract data from screenshots.

### What if I make a mistake?
- **Before submitting**: Use the Preview button to review
- **After submitting**: Admins can use the Match Editor to correct data

### How do I update existing matches?
The system detects existing matches. Click the match detection button and choose "Pre-fill Form" to update.

## Charts and Analysis

### Why don't I see any data?
- Check that a team is selected
- Verify matches exist for the selected date range
- Ensure filters aren't excluding all data

### How do I customize charts?
Click the gear icon (⚙️) on any chart to open the configuration panel. See [Chart Customization](Chart-Customization) for details.

### Can I export charts?
Export functionality may be available in future updates. Currently, charts are view-only in the browser.

### What metrics are available?
See the [Metric Glossary](Metric-Glossary) for a complete list of all available metrics and their definitions.

## Teams and Opponents

### How do I add a new team?
Team creation is available to administrators in the Settings → Teams tab.

### What if an opponent name doesn't match?
The system uses fuzzy matching to suggest similar opponent names. If no match is found, you can enter a new name.

### Can I compare multiple teams?
Yes! Use the **Club Data** view to compare multiple teams side-by-side.

## AI Chat Assistant

### How does the AI work?
The AI uses Google Gemini to understand natural language questions about your data and provide answers.

### What can I ask the AI?
You can ask about:
- Statistics and averages
- Trends and patterns
- Specific matches
- Comparisons

See [AI Chat Assistant](AI-Chat-Assistant) for examples.

### Is the AI always accurate?
The AI provides answers based on your data. Verify important information by checking charts and data views.

## Technical Questions

### Which browsers are supported?
Modern browsers including Chrome, Firefox, Safari, and Edge are supported.

### Do I need an internet connection?
Yes, JOGA Analytics is a web application that requires an internet connection.

### Is my data secure?
Yes, data is stored securely and access is controlled through user authentication and roles.

### Can I access from mobile devices?
Yes, the interface is responsive and works on tablets and mobile devices, though desktop is recommended for data entry.

## Troubleshooting

### I can't log in
- Verify your email and password
- Check for typos
- Contact your administrator if issues persist

### Charts aren't loading
- Refresh the page
- Check your internet connection
- Clear browser cache if problems persist

### Data isn't appearing
- Verify data was submitted successfully
- Check that filters aren't excluding the data
- Ensure you're viewing the correct team/date range

### Form won't submit
- Check for validation errors (red highlights)
- Ensure all required fields are filled
- Verify date format (YYYY-MM-DD)

## Getting Help

### Where can I get more help?
- Check the [Troubleshooting](Troubleshooting) guide
- Review relevant wiki pages
- Contact your administrator

Contact your system administrator for:
