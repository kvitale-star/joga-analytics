# Team Data View (Dashboard)

The Team Data view (also called Dashboard) is your primary tool for analyzing individual team performance over time.

## Overview

This view displays interactive charts showing how a selected team performs across multiple metrics. It's perfect for:
- Tracking team progress throughout a season
- Identifying trends and patterns
- Comparing performance across different time periods
- Analyzing specific metrics in detail

## Accessing Team Data View

1. Click **"Dashboard"** or **"Team Data"** in the left sidebar
2. You'll see the control bar at the top with filters
3. Charts appear below the controls

## Using Filters

### Team Selector
- **Purpose**: Select which team to analyze
- **How to Use**: 
  - Click the Team dropdown
  - Select your team from the list
  - Charts automatically update

### Opponent Filter
- **Purpose**: Filter matches by opponent
- **How to Use**:
  - Click the Opponent dropdown
  - Select one or more opponents
  - Charts show only matches against selected opponents

### Date Filter
- **Purpose**: Focus on specific time periods
- **How to Use**:
  - Click the Date dropdown
  - Select a date range or specific date
  - Charts update to show only matches in that period

### Last N Games
- **Purpose**: View recent performance
- **How to Use**:
  - Enter a number (e.g., 5, 10, 15)
  - Charts show only the most recent N games
  - Useful for analyzing current form

### Chart Group Selector
- **Purpose**: Quickly select related charts
- **How to Use**:
  - Click the Chart Group dropdown
  - Select a group (e.g., "Shooting", "Passing")
  - All charts in that group are selected

### Individual Chart Selection
- **Purpose**: Choose specific charts to display
- **How to Use**:
  - Click the Charts dropdown
  - Select/deselect individual charts
  - Multiple charts can be selected

## Understanding Charts

### Chart Types
- **Line Charts**: Show trends over time
- **Bar Charts**: Compare values across matches
- **Area Charts**: Show cumulative values

### Chart Components
- **X-Axis**: Usually represents matches (chronological order)
- **Y-Axis**: Represents the metric value
- **Data Points**: Individual match values
- **Lines/Bars**: Connect or represent data points

### Reading Charts
- **Upward Trend**: Improving performance
- **Downward Trend**: Declining performance
- **Flat Line**: Consistent performance
- **Spikes**: Exceptional games (high or low)

## Customizing Charts

### Opening Chart Configuration
1. Click the gear icon (⚙️) on any chart
2. A configuration panel opens

### Available Options

#### Metric Toggles
- Turn metrics on/off
- Example: Show "Shots For" but hide "Shots Against"
- Changes apply immediately

#### Opponent Data
- Include/exclude opponent metrics
- Example: Show both team and opponent possession
- Useful for context

#### Chart Labels
- Show/hide data labels on charts
- Helpful for precise values
- Can clutter small charts

### Saving Preferences
- Click "Save" in the configuration panel
- Your preferences are remembered
- Applies to all future sessions

## Available Charts

### Shooting Metrics
- **Shots Chart**: Shots for, shots against, goals
- **Goals Chart**: Goals scored and conceded
- **xG Chart**: Expected goals (quality of chances)
- **Conversion Rate**: Goals per shot percentage
- **Attempts Chart**: Total attempts (shots + goals)

### Possession & Passing
- **Possession Chart**: Ball possession percentage
- **Passes Chart**: Passes completed
- **Pass Share**: Percentage of total passes
- **Average Pass Length**: Average distance of passes
- **Pass Strings**: Sequences of consecutive passes

### Advanced Metrics
- **TSR Chart**: Total Shots Ratio
- **SPI Chart**: Sustained Passing Index
- **PPM Chart**: Passes Per Minute
- **Positional Attempts**: Shots by field position
- **Pass by Zone**: Passing distribution across field zones

## Tips for Effective Analysis

### 1. Start with Default View
- Begin with all default charts visible
- Get a broad overview first
- Then focus on specific metrics

### 2. Use Filters Strategically
- **Last 5 Games**: Check recent form
- **Specific Opponent**: Analyze head-to-head performance
- **Date Range**: Compare different periods (e.g., first half vs second half of season)

### 3. Compare Metrics
- Look at related metrics together
- Example: Shots + xG + Goals = complete shooting picture
- Example: Possession + Passes = passing effectiveness

### 4. Identify Patterns
- Look for trends (improving/declining)
- Identify outliers (exceptional games)
- Notice correlations (when one metric goes up, does another?)

### 5. Use Opponent Data
- Include opponent metrics for context
- Compare your performance to opponent's
- Understand relative strength

## Common Use Cases

### Pre-Game Analysis
1. Select your team
2. Filter by upcoming opponent
3. Review head-to-head statistics
4. Check recent form (Last 5-10 games)

### Post-Game Review
1. Select your team
2. Use Date filter for the specific game
3. Review all metrics for that match
4. Compare to season averages

### Season Progress Tracking
1. Select your team
2. Remove all filters (view entire season)
3. Review trends across all metrics
4. Identify areas of improvement

### Performance Comparison
1. Select your team
2. Use Date filter to compare periods
3. Example: First 10 games vs Last 10 games
4. Identify changes in performance

## Troubleshooting

### No Data Showing
- **Check**: Is a team selected?
- **Check**: Are there matches in the selected date range?
- **Check**: Do filters exclude all matches?

### Charts Not Updating
- **Try**: Refreshing the page
- **Try**: Changing and re-changing filters
- **Check**: Browser console for errors

### Missing Metrics
- **Note**: Not all metrics are available for all matches
- **Check**: If data was entered for that metric
- **Note**: Some metrics are computed from other data
