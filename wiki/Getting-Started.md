# Getting Started

Welcome to JOGA Analytics! This guide will help you get started with the platform.

## First-Time Login

1. **Access the Platform**
   - Navigate to your JOGA Analytics URL (provided by your administrator)
   - You'll see the login page

2. **Initial Setup (First User Only)**
   - If you're the first user, you'll see a setup wizard
   - Create an admin account with:
     - Your name
     - Email address
     - Secure password
   - Complete the setup process

3. **Regular Login**
   - Enter your email and password
   - Click "Login"
   - You'll be taken to the Dashboard view

## Understanding the Interface

### Navigation Sidebar
The left sidebar contains all available views:
- **Dashboard** (Team Data) - Analyze individual team performance
- **Club Data** - Compare multiple teams
- **Game Data** - View detailed match information
- **Upload Game Data** - Add new matches
- **Settings** - Manage your account
- **Metric Glossary** - Reference for all metrics
- **Match Editor** (Admin only) - Edit match records

### Main Content Area
The main area displays:
- **Control Bar** - Filters and options for the current view
- **Charts & Data** - Visualizations and statistics
- **Action Buttons** - Upload, save, and other actions

## Your First Steps

### 1. Explore the Dashboard
- Start with the **Dashboard** view (Team Data)
- Select a team from the dropdown
- View the default charts showing key metrics

### 2. Customize a Chart
- Click the gear icon (⚙️) on any chart
- Toggle metrics on/off
- Include opponent data if desired
- Click "Save" to remember your preferences

### 3. Filter Your Data
- Use the **Team** dropdown to switch teams
- Use the **Opponent** dropdown to filter by opponent
- Use the **Date** filter to focus on specific time periods
- Use **Last N Games** to see recent performance

### 4. Upload Your First Match
- Navigate to **Upload Game Data**
- Fill in the match information:
  - Team
  - Opponent
  - Date
  - Competition type
- Enter match statistics
- Click "Preview" to review
- Click "Submit" to save

## Key Concepts

### Views
Different views serve different purposes:
- **Dashboard/Team Data**: Best for tracking a single team's progress
- **Club Data**: Best for comparing teams or club-wide analysis
- **Game Data**: Best for detailed match analysis

### Charts
Charts visualize your data:
- Each chart shows specific metrics
- You can customize which metrics appear
- Charts update automatically when you change filters

### Filters
Filters help you focus on specific data:
- **Team**: Select which team to analyze
- **Opponent**: Filter matches by opponent
- **Date**: Focus on specific time periods
- **Last N Games**: View recent performance
