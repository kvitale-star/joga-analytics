# Club Data View

Compare multiple teams side-by-side and analyze club-wide performance.

## Overview

The Club Data view allows you to:
- Compare multiple teams simultaneously
- Analyze club-wide trends
- Filter by team groups (boys, girls, etc.)
- View aggregate statistics across teams

## Accessing Club Data View

1. Click **"Club Data"** in the left sidebar
2. The view opens with team comparison charts
3. Use filters to customize your analysis

## Team Selection

### Multi-Select Teams
- Click the **Teams** dropdown
- Select one or more teams to compare
- Charts update to show selected teams
- Yellow border indicates selection

### Team Groups
Use the **Options** dropdown to quickly select groups:
- **Boys Teams**: All boys teams
- **Girls Teams**: All girls teams
- **Black Teams**: All black teams
- **Show Chart Labels**: Display values on charts
- **Include Opponents**: Show opponent data

## Filters

### Teams Filter
- Select multiple teams for comparison
- Charts show all selected teams
- Useful for comparing similar teams

### Options Filter
- **Boys/Girls Teams**: Filter by gender
- **Show Chart Labels**: Toggle data labels
- **Include Opponents**: Include opponent metrics

### Last N Games
- Filter to recent games only
- Applies to all selected teams
- Useful for current form analysis

### Chart Selection
- Select which charts to display
- Multiple charts can be shown
- Customize your view

## Understanding Comparison Charts

### Multiple Lines/Bars
- Each team appears as a different color
- Legend shows team names
- Compare trends across teams

### Aggregate Views
- Some charts show club-wide totals
- Useful for overall club performance
- Combines data from all selected teams

## Use Cases

### Comparing Similar Teams
1. Select teams of similar age/level
2. Compare their performance
3. Identify best practices
4. Share insights between teams

### Club-Wide Analysis
1. Select all teams (or a group)
2. View aggregate statistics
3. Identify club-wide trends
4. Track overall progress

### Season Planning
1. Compare teams across seasons
2. Identify development patterns
3. Plan training focus areas
4. Allocate resources

## Tips

1. **Start with Similar Teams**: Compare teams of similar age/level first
2. **Use Groups**: Use the Options filter to quickly select team groups
3. **Focus on Trends**: Look for patterns across teams
4. **Compare Metrics**: Use multiple charts to get full picture
5. **Save Configurations**: Your team selections are saved
