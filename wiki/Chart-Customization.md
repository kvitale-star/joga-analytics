# Chart Customization

Learn how to customize charts to show exactly what you want to see.

## Overview

All charts in JOGA Analytics can be customized. You can:
- Toggle individual metrics on/off
- Include or exclude opponent data
- Save your preferences
- Expand charts to full width

## Accessing Chart Configuration

### Method 1: Gear Icon
1. Look for the gear icon (⚙️) on any chart
2. Click it to open the configuration panel
3. Make your changes
4. Click "Save" or "Cancel"

### Method 2: Chart Header
- Some charts have a configuration button in the header
- Click to open the same configuration panel

## Configuration Options

### Metric Toggles
Turn individual metrics on or off:

**Example - Shots Chart:**
- ✅ Shots For
- ✅ Shots Against
- ❌ Goals (hidden)
- ✅ Attempts

**How to Use:**
1. Find the metric in the list
2. Click the checkbox to toggle
3. Chart updates immediately
4. Changes are visible but not saved yet

### Opponent Data
Include or exclude opponent metrics:

**When Enabled:**
- Shows both your team's and opponent's metrics
- Useful for context and comparison
- Example: Your possession vs Opponent possession

**When Disabled:**
- Shows only your team's metrics
- Cleaner, more focused view
- Less visual clutter

**How to Toggle:**
- Check/uncheck "Include Opponent Data"
- Chart updates immediately

### Chart Labels
Show or hide data labels on the chart:

**When Enabled:**
- Numbers appear on data points
- Precise values visible
- Can clutter small charts

**When Disabled:**
- Cleaner appearance
- Values visible on hover
- Better for overview

**How to Toggle:**
- Check/uncheck "Show Chart Labels"
- Applies to all charts in the view

## Saving Preferences

### Per-Chart Preferences
1. Configure a chart
2. Click "Save" in the configuration panel
3. Your preferences are saved for that chart
4. Applies to future sessions

### Global Preferences
Some preferences apply to all charts:
- Chart Labels setting
- Opponent Data default
- Saved in your user preferences

## Expanding Charts

### Full-Width View
1. Click the expand icon (⛶) on any chart
2. Chart expands to full width
3. Better for detailed analysis
4. Click again to collapse

### Benefits of Expansion
- Larger, easier-to-read charts
- More data points visible
- Better for presentations
- Detailed trend analysis

## Custom Charts

### Creating Custom Charts
1. Navigate to **Settings** → **Custom Charts**
2. Click "Create Custom Chart"
3. Select metrics to include
4. Choose chart type (line, bar, area, scatter)
5. Configure axes and options
6. Save your custom chart

### Using Custom Charts
1. Custom charts appear in chart selectors
2. Select them like any other chart
3. They respect your filters
4. Can be customized further

**Note:** Custom charts are a Phase 1 feature. See [User Agency Features](User-Agency-Features) for more details.

## Chart Types

### Line Charts
- Best for: Trends over time
- Shows: Continuous data progression
- Use when: Tracking changes across matches

### Bar Charts
- Best for: Comparing values
- Shows: Discrete match comparisons
- Use when: Comparing specific games

### Area Charts
- Best for: Cumulative values
- Shows: Stacked or filled areas
- Use when: Showing totals or portions

## Tips for Effective Customization

### 1. Start with Defaults
- Begin with default chart configurations
- Get familiar with the data
- Then customize based on your needs

### 2. Focus on Key Metrics
- Don't show too many metrics at once
- Focus on 2-4 related metrics per chart
- Reduces visual clutter

### 3. Use Opponent Data Strategically
- Include when comparing performance
- Exclude when focusing on your team
- Toggle based on your analysis goal

### 4. Save Frequently Used Configurations
- If you always customize charts the same way, save it
- Saves time in future sessions
- Consistent analysis approach

### 5. Experiment
- Try different configurations
- See what works best for your analysis
- Don't be afraid to change things

## Examples

### Example 1: Shooting Analysis
**Goal:** Analyze shooting performance

**Configuration:**
- ✅ Shots For
- ✅ Goals
- ✅ xG
- ❌ Shots Against (hidden)
- ✅ Include Opponent Data (for context)

**Result:** Clear view of your shooting effectiveness with opponent context

### Example 2: Possession Focus
**Goal:** Focus only on possession

**Configuration:**
- ✅ Possession
- ❌ All other metrics (hidden)
- ❌ Opponent Data (excluded)
- ✅ Show Labels

**Result:** Clean, focused possession chart with precise values

### Example 3: Comparison View
**Goal:** Compare team vs opponent

**Configuration:**
- ✅ Shots For
- ✅ Shots Against
- ✅ Goals For
- ✅ Goals Against
- ✅ Include Opponent Data

**Result:** Side-by-side comparison of key metrics

## Resetting to Defaults

### Per-Chart Reset
1. Open chart configuration
2. Click "Reset to Defaults"
3. Chart returns to original configuration
4. Your saved preferences are cleared

### Global Reset
1. Navigate to **Settings** → **Preferences**
2. Find "Reset Chart Preferences"
3. Click to reset all charts
4. All charts return to defaults

## Troubleshooting

### Changes Not Saving
- **Check**: Did you click "Save"?
- **Check**: Are you logged in?
- **Try**: Refreshing the page

### Chart Not Updating
- **Try**: Closing and reopening configuration
- **Try**: Refreshing the page
- **Check**: Are filters excluding all data?

### Preferences Lost
- **Note**: Preferences are saved per user
- **Check**: Are you logged in as the same user?
- **Note**: Browser cache may affect saved preferences
