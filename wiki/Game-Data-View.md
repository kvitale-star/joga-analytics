# Game Data View

View and analyze detailed match-by-match data.

## Overview

The Game Data view provides:
- Detailed statistics for individual matches
- Match-by-match comparison
- Filterable match list
- Metric selection and customization

## Accessing Game Data View

1. Click **"Game Data"** in the left sidebar
2. You'll see filters at the top
3. Match data appears below

## Filters

### Team Filter
- Select a team to view their matches
- Required for viewing matches
- Yellow border when selected

### Opponent Filter
- Filter matches by opponent
- Multi-select available
- Compare performance against specific opponents

### Date Filter
- Filter by date range
- Focus on specific periods
- Compare different timeframes

## Match List

### Match Cards
Each match displays:
- Match date
- Opponent name
- Key statistics
- Result (if available)

### Sorting
- Matches sorted by date (newest first)
- Chronological order
- Easy to find recent games

## Metric Selection

### Category Filters
Filter metrics by category:
- **Shooting**: Shots, goals, xG, conversion rates
- **Passing**: Passes, pass share, pass strings
- **Possession**: Possession percentage
- **JOGA Metrics**: SPI, TSR, and other advanced metrics
- **Defense**: Defensive statistics
- **Set Pieces**: Corners, free kicks

### Metric Display
- Selected metrics appear in match cards
- Toggle metrics on/off
- Customize your view
- Preferences are saved

## Analyzing Matches

### Individual Match Analysis
1. Select a team
2. Find the match in the list
3. Review all displayed metrics
4. Compare to other matches

### Trend Analysis
1. Select a team
2. Review matches chronologically
3. Identify trends
4. Spot patterns

### Opponent Analysis
1. Select a team
2. Filter by specific opponent
3. Review head-to-head performance
4. Identify strengths/weaknesses

## Tips

1. **Use Filters**: Combine team, opponent, and date filters
2. **Select Relevant Metrics**: Focus on metrics important to your analysis
3. **Compare Matches**: Look for patterns across multiple games
4. **Check Trends**: Review matches in chronological order
5. **Save Preferences**: Your metric selections are remembered
