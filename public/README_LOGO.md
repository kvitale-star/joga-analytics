# JOGA Logo

Please place your JOGA club logo image file here as `joga-logo.png`.

The logo should be:
- A square image (recommended: 64x64px or 128x128px for best quality)
- PNG format with transparency (or white background)
- The bright lime green JOGA logo with circular border

The Sidebar component will automatically:
- Display the logo when the menu is collapsed
- Show the logo + "JOGA" text when the menu is expanded

If the image file is not found, it will fall back to showing "J" or "JOGA" text.
